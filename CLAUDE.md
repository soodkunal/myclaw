# CLAUDE.md - MyClaw Developer Guidelines

## Build Commands
- Initialize Web: `cd website && npm run dev`

## Styling & Libraries
- We are using **Next.js** with **shadcn/ui** and **RetroUI** (`https://retroui.dev`).
- Ensure all components are styled using Tailwind CSS according to the neo-brutalist aesthetics defined by RetroUI.

## Code Quality
- Enforce strict typing in all TypeScript files.
- Keep the `skills/` and `agents/` directories organized with clear English-only documentation adjacent to the code.
- General guides and `README.md` must eventually support English, French, German, Dutch, and Spanish.
