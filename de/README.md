# MyClaw 🦖

> Das ultimative Wissenssystem für Claude Code

MyClaw ist eine Open-Source-Sammlung von praxiserprobten Mustern, Skills, Agenten, Hooks, Workflows und Prompts, die entwickelt wurde, um Ihre Erfahrung mit dem Anthropic Claude Code CLI zu maximieren. Integrieren Sie dies in Ihre Umgebung, um Ihren Programmier-Output ohne Installationsaufwand zu vervielfachen.

## Funktionen vs. Vanilla Claude Code

- **Erweiterte Sprachunterstützung**: Vorkonfigurierte Skills für Python (FastAPI/Django), Go, C#/.NET, Next.js.
- **Tiefgreifende DevOps-Tools**: Spezialisierte Agenten und Skills für Kubernetes, Docker, Terraform und GitHub Actions.
- **Vollständige End-to-End-Workflows**: Mehrstufige Markdown-Workflows zur Anleitung von Claude bei komplexen Debugging- oder Refactoring-Sitzungen.
- **Sicherheit & Hooks**: Vorkonfigurierte `settings.json`-Hooks zur Token-Optimierung, Sicherheit und Git-Automatisierung.

## Schnellstart

Sie können das komplette System über NPX installieren:

```bash
npx myclaw init
```

Dadurch werden die `.claude/`-Tools, -Skills und -Einstellungen in Ihr aktuelles Arbeitsverzeichnis kopiert.

## Manuelle Installation

Alternativ können Sie auch nur das kopieren, was Sie benötigen:

1. Klonen Sie dieses Repository: `git clone https://github.com/yourusername/myclaw.git`
2. Kopieren Sie die gewünschten `skills/`- oder `agents/`-Dateien in das `.claude/`-Verzeichnis Ihres Projekts.
