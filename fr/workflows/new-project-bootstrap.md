# New Project Bootstrap Workflow

Starting a project from scratch? Let Claude do the heavy lifting.

## Étape 1: Architecture Plan
Invoke the **Architect** agent:
> "I want to build a SaaS application with Next.js, FastAPI, and PostgreSQL. Use the Architect agent to write an `ARCHITECTURE.md` file detailing the tech stack and database schema."

## Étape 2: Scaffolding
Generate the boilerplate:
> "Initialize the Next.js frontend using `npx create-next-app` and set up a basic FastAPI backend structure."

## Étape 3: CI/CD Setup
Set up standard DevOps practices:
> "Create a `Dockerfile` for the backend and a `.github/workflows/ci.yml` file to run tests on push."
