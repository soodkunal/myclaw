# Next.js Frontend Reference

Include this file in your Next.js project root to load the necessary MyClaw rules and skills.

## Architecture
- Framework: Next.js (App Router)
- Styling: TailwindCSS + shadcn/ui
- Language: TypeScript

## Plugin Loading
Ensure your `plugin.json` loads the `typescript`, `nextjs`, and `npm-resolver` modules.
