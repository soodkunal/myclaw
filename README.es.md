# MyClaw 🦖

> El sistema de conocimiento definitivo para Claude Code

MyClaw es una colección de código abierto de patrones, habilidades, agentes, ganchos (hooks), flujos de trabajo y prompts probados en batalla, diseñada para potenciar tu experiencia con la CLI de Anthropic Claude Code. Integra esto en tu entorno para multiplicar tu rendimiento de codificación sin fricción de instalación.

## Características frente a Vanilla Claude Code

- **Soporte ampliado de lenguajes**: Habilidades preconfiguradas para Python (FastAPI/Django), Go, C#/.NET, Next.js.
- **Herramientas DevOps profundas**: Agentes y habilidades especializadas para Kubernetes, Docker, Terraform y GitHub Actions.
- **Flujos de trabajo completos de extremo a extremo**: Flujos de trabajo en markdown de múltiples pasos para guiar a Claude a través de sesiones complejas de depuración o refactorización.
- **Seguridad y Hooks**: Hooks preconfigurados en `settings.json` para la optimización de tokens, seguridad y automatización de git.

## Inicio Rápido

Puedes instalar el sistema completo a través de NPX:

```bash
npx myclaw init
```

Esto copiará las herramientas, habilidades y configuraciones de `.claude/` a tu directorio de trabajo actual.

## Instalación Manual

Alternativamente, puedes simplemente copiar lo que necesites:

1. Clona este repositorio: `git clone https://github.com/yourusername/myclaw.git`
2. Copia los archivos deseados de `skills/` o `agents/` directamente en el directorio `.claude/` de tu proyecto.
